## 使用说明

1. **生成图片**：使用 `插图AI提示词清单.md` 中的提示词生成图片
2. **命名规范**：严格按照上述文件名命名
3. **放置图片**：将生成的图片放入对应文件夹
4. **格式要求**：
   - 格式：PNG
   - 尺寸：1920x1080px (16:9)
   - 文件大小：不超过2MB
