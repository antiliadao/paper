## 排版规范

### 标题层级
```markdown
# 章节标题
## 主要段落
### 子段落
#### 细节说明
```

### 强调文本
- **重要概念** - 使用加粗
- *次要强调* - 使用斜体
- `代码/术语` - 使用行内代码

### 列表格式
- 使用破折号(-)作为无序列表
- 保持层级缩进一致
- 每项内容简洁明了

### 引用块
> 用于重要观点或名言引用
> 保持简洁有力

### 分隔线
使用三个破折号 --- 创建分隔线

## 内容结构

1. **开篇引言** - 场景化描述，引入主题
2. **核心论述** - 层次分明，逻辑清晰
3. **案例说明** - 具体实例，便于理解
4. **总结提炼** - 要点回顾，价值强调

## 视觉优化

- 去除所有emoji表情
- 使用Lucide图标组件
- 保持简洁专业的风格
- 适度使用强调和装饰
